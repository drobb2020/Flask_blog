Modules Required for this Website

The following Python modules are required for this website to function.

  • Flask
  • Flask-Bcrypt
  • Flask-Login
  • Flask-Mail
  • Flask-SQLAlchemy
  • Flask-WTF
  • itsdangerous
  • Jinja2
  • SQLAlchemy
  • WTForms

    • ** and supporting modules for those listed

Along with these modules you also require:

  • Bootstrap 4.0.0
  • jquery 3.2.1
  • Popper.js 1.12.9
